# 🧠 AI Fact Checker

A machine learning + NLP-based web app that verifies news claims using real-time articles and semantic similarity.

---

## 🚀 Features
- Real-time news fetching (NewsAPI)
- AI-based semantic similarity (Sentence Transformers)
- Fake vs Real evidence scoring
- Clean modern UI
- Flask web app

---

## 🛠 Tech Stack
- Python
- Flask
- Scikit-learn
- Sentence Transformers
- NewsAPI

---

## 📸 Demo
Enter a news claim → Get AI verdict with confidence score

---

```bash
pip install -r requirements.txt
